# Querying the data

**The basics:**

```JavaScript
db.alunos.find()
db.alunos.find().pretty() // Beautify results
```


## Specifying the query

**By name**
```JavaScript
db.alunos.find({
  nome: 'Victor'
}).pretty()
```

**By skill (making a "join")**
```JavaScript
db.alunos.find({
  'nome': 'Victor',
  'habilidades.nome': 'inglês'
}).pretty()
```

**The "Or" clause**
```JavaScript
db.alunos.find({
  $or: [
    { 'curso.nome': 'Tecnologia em Sistemas para Internet' },
    { 'curso.nome': 'Medicina' }
  ]
}).pretty()
```

**The "Or" clause with more parameters**
```JavaScript
db.alunos.find({
  $or: [
    { 'curso.nome': 'Tecnologia em Sistemas para Internet' },
    { 'curso.nome': 'Medicina' },
    { 'curso.nome': 'Engenharia da Computação' }
  ],
  'nome': 'Daniela' // Work like an AND
}).pretty()
```

**The "In" clause like SQL**
```JavaScript
db.alunos.find({
  'curso.nome': {
    $in: [
      'Tecnologia em Sistemas para Internet',
      'Medicina',
    ]
  }
}).pretty()
```
